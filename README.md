# Playlist UI

A simple and responsive music playlist interface built with **HTML** and **CSS**.It contains all the songs i listen to while i travel to college.
This project focuses on clean layout, styling, and basic UI elements to showcase frontend design skills.

## Features

- Responsive layout suitable for desktop and mobile
- Clean and modern design
- Playlist items with title and artists
- Hover effects on playlist items for interactivity
- Fully built with HTML and CSS (no JavaScript)
